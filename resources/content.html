Add Sublime Settings Folder
Select your Sublime settings folder here. Note that this is *NOT* the Sublime installation folder.
The Sublime Settings folder can usually be found at this places:
Windows: %userprofile%\AppData\Roaming\Sublime Text 3
macOSX: Library/Application Support/Sublime Text 3
Linux: .config/sublime-text-3
